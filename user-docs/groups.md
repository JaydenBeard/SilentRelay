# Groups

Create and manage group conversations with the same end-to-end encryption as private messages.

## Creating a Group

### Basic Group Setup

1. **Start Group Creation**
   - Go to Settings â†’ Groups
   - Click "Create New Group"
   - Or use the "+" menu in conversations list

2. **Group Details**
   - **Name**: Choose a clear, descriptive name (max 100 characters)
   - **Description**: Optional description of the group's purpose
   - **Avatar**: Upload a group photo or use default icon

3. **Add Members**
   - Search for users by username
   - Add up to 100 members at creation
   - Members can be added later by admins

4. **Group Settings**
   - Choose privacy level (public/open or private/invite-only)
   - Set initial permissions

### Group Types

**Public Groups**:
- Anyone can find and join
- Listed in group discovery
- Suitable for communities

**Private Groups**:
- Invite-only membership
- Not visible in searches
- Better for private discussions

## Managing Group Members

### Adding Members

**As Admin**:
1. Open group chat
2. Tap group name/avatar
3. Select "Add Members"
4. Search and select users
5. Send invitations

**Member Permissions**:
- New members receive invitation
- Must accept to join
- Group history becomes visible

### Removing Members

**Admin Actions**:
1. Open group settings
2. Go to "Members" tab
3. Find the member
4. Select "Remove from Group"
5. Confirm removal

**Effects of Removal**:
- Member loses access to group
- Messages remain in history
- Can be re-invited later

### Member Roles

**Group Creator**:
- Full administrative control
- Can dissolve the group
- Cannot be removed by others

**Group Admins**:
- Add/remove members
- Change group settings
- Moderate content

**Regular Members**:
- Send messages
- View group content
- Leave group voluntarily

## Group Security

### End-to-End Encryption

**Same as Private Chats**:
- All messages encrypted with Signal Protocol
- Group keys shared securely
- Perfect forward secrecy maintained

**Group Key Management**:
- Keys rotate when membership changes
- Protects against compromised members
- Automatic re-encryption for security

### Safety Numbers

**Group Verification**:
- Each member has individual safety numbers
- Verify admins and key members
- Group safety number changes with membership

**Verification Process**:
1. Open group chat
2. Tap group name
3. View "Group Safety Numbers"
4. Compare with other members

### Messaging

**Group Chat Features**:
- Send text, emoji, files
- @mentions for specific members
- Reply to specific messages
- Message reactions

**Advanced Features**:
- Disappearing messages (group-wide setting)

### File Sharing

**Group File Sharing**:
- Same 100MB limit per file
- All files encrypted

- Up to 8 participants

### Moderation

**Admin Tools**:

## Managing Groups

### Group Administration

**Settings Access**:
1. Open group chat
2. Tap group name/avatar
3. Select "Group Settings"

**Available Controls**:
- Change group name/description/avatar
- Manage members and roles
- Adjust privacy settings
- View group statistics

### Group Analytics

**Available Metrics**:
- Member count and activity
- Message volume
- File storage usage
- Peak activity times

### Dissolving Groups

**Permanent Action**:
1. Only group creator can dissolve
2. All members notified
3. Group becomes inaccessible
4. Messages archived (if enabled)

**Recovery**:
- Dissolved groups cannot be restored
- Members keep personal message history
- No data recovery possible

## Group Best Practices

### For Group Admins

**Security**:
- Verify member identities for sensitive groups
- Use disappearing messages for temporary discussions
- Regularly review member list
- Set appropriate privacy levels

**Management**:
- Clear group purpose and rules
- Assign multiple admins
- Moderate actively in large groups
- Archive inactive groups

### For Members

**Participation**:
- Respect group guidelines
- Use @mentions appropriately
- Keep discussions on-topic
- Report inappropriate content

**Privacy**:
- Be cautious sharing personal information
- Verify group safety numbers
- Leave groups you no longer need

## Troubleshooting Groups

### Can't Create Group

**Possible Issues**:
- Insufficient permissions
- Server limits reached
- Account verification required

**Solutions**:
- Check account status
- Contact administrator
- Try again later

### Messages Not Sending

**Group-Specific Issues**:
- Check if you're still a member
- Verify group hasn't been dissolved
- Try leaving and rejoining

### Member Can't Join

**Invitation Issues**:
- Check invitation hasn't expired
- Verify username spelling
- Member may have blocked invites

### Group Not Appearing

**Visibility Problems**:
- Check if group is private
- Verify you're still a member
- Try logging out and back in

## Group Limits and Quotas

### Size Limits

- **Members**: 100 maximum
- **Messages**: No limit (server-dependent)
- **Files**: 100MB per file, total storage quota
- **Name Length**: 100 characters

### Rate Limits

- **Message Frequency**: No strict limits
- **File Uploads**: Rate limited for abuse prevention
- **Member Changes**: Admin actions rate limited

## Privacy Considerations

### Data Visibility

**What Members Can See**:
- All group messages
- Member list (if not hidden)
- Group settings and rules
- File attachments

**What Admins Can See**:
- Additional moderation tools
- Member activity statistics
- Hidden settings and controls

### Leaving Groups

**Clean Exit**:
1. Open group settings
2. Select "Leave Group"
3. Confirm action

**After Leaving**:
- Lose access to group
- Keep personal message history
- Can be re-invited by admins

Remember: Group conversations are only as secure as their most vulnerable member. Choose participants carefully and follow security best practices.
