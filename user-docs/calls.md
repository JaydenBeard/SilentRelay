# Voice & Video Calls

Make secure, end-to-end encrypted voice and video calls in SilentRelay.

## Starting Calls

### Voice Calls

1. **Open Chat**
   - Navigate to any conversation
   - Click the phone icon in chat header

2. **Call Initiation**
   - Call rings on recipient's device
   - Shows caller information
   - Recipient can accept, decline, or ignore

3. **During Call**
   - Mute/unmute with microphone button
   - Speakerphone toggle (mobile)
   - Call duration displayed

### Video Calls

1. **Start Video**
   - Click camera icon in chat header
   - Grants camera/microphone permissions

2. **Video Controls**
   - Camera on/off toggle
   - Switch between front/rear camera (mobile)
   - Full screen mode

3. **Screen Sharing**
   - Share your screen during video calls
   - Select entire screen, window, or tab
   - Recipient sees shared content

## Call Features

### Audio Controls

- **Mute**: Temporarily disable microphone
- **Speaker**: Toggle speakerphone (mobile)
- **Volume**: Adjust call volume
- **Noise Cancellation**: Automatic background noise reduction

### Video Controls

- **Camera Toggle**: Turn video on/off during call
- **Camera Switch**: Front/rear camera on mobile
- **Full Screen**: Expand video to full screen
- **Picture-in-Picture**: Continue call while using other apps

### Advanced Features

- **Call Recording**: Not supported (privacy protection)
- **Call Transfer**: Transfer to another contact

- **Call History**: View past call logs

## Call Security

### End-to-End Encryption

**Call Encryption**:
- All calls use WebRTC with DTLS-SRTP
- Encryption keys negotiated securely
- No server access to call content
- Perfect forward secrecy

**Metadata Protection**:
- Call timing visible to server
- Participant list hidden (sealed sender)
- No call content logging

### Verification

**Call Safety**:
- Verify participants through safety numbers
- End-to-end encrypted indicator
- No man-in-the-middle possible

## Troubleshooting Calls

### Call Won't Connect

**Common Issues**:
- **Network Problems**: Check internet connection
- **Permissions**: Grant camera/microphone access
- **Device Busy**: End other calls first
- **Server Issues**: Check service status

**Solutions**:
1. Refresh the page
2. Check browser permissions
3. Try different browser
4. Contact support

### Poor Audio/Video Quality

**Quality Issues**:
- **Bandwidth**: Low internet speed
- **Network Congestion**: Too many devices on network
- **Device Performance**: Old/slow device
- **Background Apps**: Close unnecessary applications

**Improvements**:
1. Use wired connection
2. Close other applications
3. Reduce video quality
4. Try voice-only call

### Echo or Feedback

**Audio Problems**:
- **Acoustic Feedback**: Speaker too close to microphone
- **Device Issues**: Faulty microphone/speaker
- **Multiple Devices**: Calls on multiple devices

**Fixes**:
1. Use headphones
2. Adjust speaker volume
3. Mute when not speaking
4. Test microphone separately

### Camera Not Working

**Camera Issues**:
- **Permissions**: Browser blocked camera access
- **Hardware**: Camera faulty or disabled
- **Driver Issues**: Outdated camera drivers
- **Multiple Apps**: Camera used by another app

**Solutions**:
1. Check browser permissions
2. Restart browser
3. Test camera in other apps
4. Update device drivers
