# Getting Started with SilentRelay

Welcome to SilentRelay! This guide will help you get started with secure, private messaging in just a few minutes.

## Creating Your Account

1. **Visit SilentRelay**
   - Open your web browser and go to your SilentRelay instance
   - Click "Sign In" or "Create Account"

2. **Choose a Username**
   - Pick a unique username (3-30 characters)
   - Usernames are public and used for finding contacts
   - You can change your display name later in settings

3. **Set Up Security**
   - Create a 4 or 6-digit PIN for quick login
   - This PIN protects your account on this device
   - Write down your 24-word recovery key and store it safely
   - The recovery key can restore your account if you lose access

4. **Verify Your Account**
   - Check your email/SMS for a verification code
   - Enter the code to complete registration

## Your First Message

1. **Start a New Chat**
   - Click the "+" button or "Start New Chat"
   - Search for a contact by typing their username (minimum 3 characters)
   - Select the user from the search results

2. **Send a Message**
   - Type your message in the input field
   - Press Enter or click Send
   - Your message is automatically encrypted before sending

3. **Verify Your Contact (Recommended)**
   - Tap the contact's name in the chat header
   - Go to "Safety Number" or "Verify Contact"
   - Compare safety numbers in person or via another channel
   - This ensures you're talking to the right person

## Understanding the Interface

### Main Screen
- **Conversation List**: Shows all your chats, sorted by recent activity
- **Chat View**: Displays messages in the selected conversation
- **Message Input**: Type and send messages, attach files

### Navigation
- **Desktop**: Side-by-side layout with conversation list and chat
- **Mobile**: Swipe between conversation list and chat view

### Status Indicators
- **Green dot**: Contact is online
- **Check marks**: Message delivery status
  - Sent
  - Delivered
  - Read

## Basic Settings

Access settings by clicking your profile picture or the gear icon:

- **Profile**: Change display name, avatar, status
- **Privacy**: Control read receipts, last seen, disappearing messages
- **Notifications**: Manage push notifications and sounds
- **Security**: View linked devices, change PIN
