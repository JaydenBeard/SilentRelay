## Messaging

### Starting Conversations

1. **New Chat**
   - Click "Start New Chat" or the "+" button
   - Search for users by username (type at least 3 characters)
   - Select from search results to start chatting

2. **Existing Conversations**
   - Click any conversation in the sidebar to open it
   - Conversations are sorted by most recent activity

### Sending Messages

- **Text Messages**: Type and press Enter
- **Emoji**: Use your device's emoji keyboard
- **Line Breaks**: Shift+Enter for new lines
- **Editing**: Messages cannot be edited after sending (for security)

### Message Status

- **Sending**: Message is being encrypted and sent
- **Sent**: Message delivered to server
- **Delivered**: Message reached recipient's device
- **Read**: Recipient has opened the chat

### Typing Indicators

- See when someone is typing (can be disabled in privacy settings)
- Your typing status is shared with the current chat only

## Groups

### Creating Groups

1. Go to Settings → Groups
2. Click "Create New Group"
3. Add a group name and description
4. Invite members by username
5. Set group permissions

### Group Features

- **Admin Controls**: Group creators can add/remove members
- **Member Limits**: Up to 100 members per group
- **Group Info**: Change name, avatar, description
- **Leave Group**: Members can leave at any time

### Group Security

- Same end-to-end encryption as 1-on-1 chats
- All members must verify the group (safety numbers)
- Group keys are rotated when members change

## File Sharing

### Sending Files

1. Click the paperclip icon in message input
2. Select files from your device
3. Files are encrypted before upload
4. Maximum file size: 100MB

### Supported Formats

- Images: JPG, PNG, GIF, WebP
- Documents: PDF, DOC, DOCX, TXT
- Audio: MP3, WAV, M4A
- Video: MP4, MOV, AVI
- Archives: ZIP, RAR

### Downloading Files

- Click any file attachment to download
- Files are decrypted on your device only
- Downloads work offline once cached

## Voice & Video Calls

### Starting Calls

1. Open a chat
2. Click the phone icon (voice) or camera icon (video)
3. Wait for the other person to answer

### Call Features

- **End-to-End Encrypted**: Calls are fully encrypted
- **Screen Sharing**: Share your screen during video calls
- **Mute/Unmute**: Control your audio
- **Camera Toggle**: Turn video on/off
- **Call Recording**: Not supported (privacy protection)

### Call Quality

- **WebRTC**: Industry-standard for real-time communication
- **Adaptive Quality**: Automatically adjusts to connection
- **Noise Cancellation**: Built-in audio processing

## Privacy Settings

### Read Receipts

- **Send Read Receipts**: Let others know when you read their messages
- **Receive Read Receipts**: See when others read your messages
- Default: Enabled

### Last Seen

- **Show Last Seen**: Display when you were last active
- **Hide Last Seen**: Keep your activity private
- Default: Show to contacts

### Disappearing Messages

- **Enable**: Messages auto-delete after selected time
- **Timer Options**: 5s, 10s, 30s, 1m, 5m, 1h, 1d, 1w
- **Per Chat**: Set different timers for different conversations

### Blocked Contacts

- Block users to stop receiving messages
- Blocked users can't see your online status
- You won't receive notifications from blocked users

## Device Management

### Linking Devices

1. Go to Settings → Devices
2. Click "Link New Device"
3. Scan QR code with new device
4. Devices share the same encryption keys

### Device Security

- **Device Names**: Give descriptive names to your devices
- **Last Active**: See when each device was last used
- **Remove Device**: Deauthorize devices you no longer own

### Security Best Practices

- Only link devices you control
- Remove old devices immediately when lost/stolen
- Use strong PINs on all devices

## Notifications

### Push Notifications

- **Enable/Disable**: Control all notifications
- **Per Chat**: Mute specific conversations
- **Preview**: Show/hide message preview in notifications

### Notification Types

- **New Messages**: Alerts for incoming messages
- **Calls**: Ring for incoming calls
- **Group Activity**: Notifications for group events
- **Security Alerts**: Important security notifications

### Do Not Disturb

- **Schedule**: Set quiet hours
- **Exceptions**: Allow calls from important contacts

## Account Settings

### Profile

- **Display Name**: Your public name (can be changed)
- **Username**: Unique identifier (cannot be changed)
- **Avatar**: Profile picture
- **Status**: Custom status message

### Security

- **Change PIN**: Update your device PIN
- **Recovery Key**: View/download your 24-word backup
- **Two-Factor Auth**: Additional verification (if enabled)

### Data Management

- **Export Data**: Download all your messages and media
- **Delete Account**: Permanently remove your account
- **Clear Cache**: Free up storage space

## Advanced Features

### Safety Numbers

- Unique fingerprints for each contact
- Verify in person for maximum security
- Changes if someone tries to intercept

### Key Transparency

- Blockchain-like verification of encryption keys
- Prevents key substitution attacks
- Automatic background verification

### Offline Messaging

- Messages sent while offline are delivered when you reconnect
- File uploads resume automatically
- No message loss due to connectivity issues
