# Migration & Upgrade Guide

Learn how to migrate from other messaging apps and handle SilentRelay updates.

## Migrating from Other Apps

### From Signal

**Data Export**:
1. Open Signal Desktop
2. Go to Settings → Chats → Export chats
3. Export conversations as HTML or TXT
4. Save exported files securely

**Contact Migration**:
- Signal contacts must be added manually by username
- No automatic contact import (privacy protection)
- Share your SilentRelay username with contacts

**Key Differences**:
- Username-based instead of phone numbers
- Self-hostable server option
- More advanced group controls

### From WhatsApp

**Export Process**:
1. Open WhatsApp → Settings → Chats → Export Chat
2. Choose chats to export
3. Export with or without media
4. Transfer files to SilentRelay device

**Migration Steps**:
1. Create SilentRelay account
2. Share username with WhatsApp contacts
3. Import important chat history manually
4. Set up groups in SilentRelay

**Security Note**: WhatsApp uses different encryption - verify safety numbers in SilentRelay

### From Telegram

**Data Export**:
- Telegram allows chat export through desktop app
- Export individual chats or all chats
- Include media files if needed

**Migration Considerations**:
- Telegram has bot features SilentRelay doesn't have
- SilentRelay focuses on privacy over features
- Self-destructing messages available in both

## Account Migration

### Moving Between Servers

**If switching self-hosted instances**:

1. **Export Data**
   - Settings → Privacy → Export Data
   - Download all messages and media
   - Save encryption keys securely

2. **Create New Account**
   - Register on new server
   - Use same username if available
   - Restore from backup if supported

3. **Transfer Contacts**
   - Share new server information
   - Re-verify safety numbers
   - Update groups and settings

### Device Migration

**Switching Devices**:
1. Link new device to account
2. Wait for message sync
3. Test functionality
4. Remove old device

**Recovery Process**:
- Use 24-word recovery key
- Available for account recovery
- Store securely offline

## SilentRelay Updates

### Automatic Updates

**Web Version**:
- Updates automatically in browser
- No user action required
- New features appear automatically

**Mobile Apps**:
- Update through app stores
- Manual update recommended
- Check for updates regularly

### Major Version Updates

**Breaking Changes**:
- Check release notes before updating
- Backup important data
- Test on secondary device first

**New Features**:
- Review changelog for new capabilities
- Update privacy settings as needed
- Re-verify safety numbers after major updates

## Backup and Recovery

### Creating Backups

**Regular Backups**:
- Export data monthly
- Save recovery key securely
- Backup encryption keys
- Document important settings

**Backup Locations**:
- Encrypted external drive
- Password manager
- Secure cloud storage (encrypted)
- Physical safe for recovery key

### Recovery Process

**Account Recovery**:
1. Use recovery key on new device
2. Restore messages from backup
3. Re-link other devices
4. Re-verify all contacts

**Data Recovery**:
- Import exported conversations
- Restore media files
- Recreate groups if needed
- Update contact information

## Troubleshooting Migration

### Common Issues

**Messages Not Syncing**:
- Check internet connection
- Wait for sync completion
- Refresh browser/app
- Contact support if persistent

**Contacts Not Found**:
- Verify usernames are correct
- Check if accounts exist
- Try different search terms

**File Import Failures**:
- Check file format compatibility
- Reduce file sizes if needed
- Try importing in batches

### Data Loss Prevention

**Before Migration**:
- Export all important data
- Verify backups are complete
- Test recovery process
- Document custom settings

**During Migration**:
- Don't delete old accounts immediately
- Keep both apps running temporarily
- Test SilentRelay thoroughly first

## Security Considerations

### Key Management

**During Migration**:
- Safety numbers change when switching apps
- Re-verify all contacts
- Be suspicious of number changes
- Use secure channels for verification

### Privacy Settings

**After Migration**:
- Review all privacy settings
- Enable disappearing messages if desired
- Configure notification preferences
- Set up device management
